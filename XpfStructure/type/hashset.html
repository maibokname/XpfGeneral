HashSet

type, public, extends(BaseIterable) :: HashSet

The HashSet type is a collection type that employs an open-addressing hash table implementation to provide common operations for an unordered set. It makes no guarantees as to the iteration order of the set; in particular, it does not guarantee that the order will remain constant over time.
