LongRNG

type, public, ABSTRACT, extends(BaseRNG) :: LongRNG

The LongRNG type is an abstract PRNG type that directly extends the BaseRNG type. It is provided to aid the implementation of an Long PRNG whose main purpose is to produce a 64-bit integer random number. Therefore, all so-called Long PRNGs should extend from this base type.
