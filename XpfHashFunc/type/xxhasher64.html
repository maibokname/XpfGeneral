XXHasher64

type, public, extends(Hasher64) :: XXHasher64

XXHasher64 is a hasher type that outputs the hash value as a 64-bit integer. It employs the XX hash algorithm by Yann Collet.

        PRIVATE

        tUInt64     :: State(4)             = 0_kInt64
        !% buffer array used to store input data

        tLogical    :: RemoveSign   = FalseVal
    CONTAINS
