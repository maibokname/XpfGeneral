PURPOSE OF THIS MODULE:
This module contains various utility routines relating to writing to character strings (i.e. formatting a value).


Uses

    Arguments

    Type IntentOptional Attributes Name
    real(kind=kSingle), intent(in) :: Number

    number
    character(kind=kChar, len=20), intent(inout) :: StrBuf

    string buffer
    integer(kind=kInt32), intent(in), optional :: B2DAlgo

    flag for binary-to-decimal algorithm
    - B2D_DragonBox (or 1) if using the DragonBox algorithm.
    - B2D_Ryu (or 2) if using the Ryu algorithm.
    - B2D_Schubfach (or 3) if using the Schubfach algorithm.
    - Default is B2D_DragonBox.
    logical, intent(in), optional :: IsScientific

    format flag
    - True if to write the given number in scientific format.
    - False if to write the given number in general format.
    - Default is false.
    integer(kind=kInt32), intent(in), optional :: MaxD

    (maximum) number of digits to be written after the decimal point.

    character(kind=kChar, len=*), intent(in), optional :: Fmt

    format to be used in the WRITE statement for the specified number.
    If present, the procedure does not check its validity and will try to perform an internal write of the specified number based on the specified format while ignoring all other optional input. If success, the procedure return immediately. Otherwise, the procedure will check other optional input and write the number accordingly.

    Return Value integer(kind=kInt32)

    length of the written decimal string
