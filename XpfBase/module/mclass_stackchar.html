StackChar

PURPOSE OF THIS MODULE:
This module contains the StackChar type and its related/supporting routines and derived type(s). The StackChar type is a stack container with CHARACTER as the type of its stored items. It employs a conventional singly-linked list implementation.


Uses

    • MBase_ErrHandlers
    • MBase_Common
    • MBase_MemHandlers

Derived Types

type, public ::  StackChar

StackChar is a container type that employs a singly-linked list implementation to provide common operations for a LIFO stack container. The type of items stored in this container is CHARACTER.
