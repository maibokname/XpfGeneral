PURPOSE OF THIS MODULE:
This module contains the ErrorInfo derived type and its related routines. The ErrorInfo type is an error data type that can be used to handle and report errors. Unlike the ErrorData type, which can only report the routine name (and optionally the module name) where the error occurs, the ErrorInfo type provides the SetCaller method so that all the calling routines and their modules can be reported.
