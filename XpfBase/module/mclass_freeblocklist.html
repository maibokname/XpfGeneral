FreeBlockNode FreeBlockList

PURPOSE OF THIS MODULE:
This module contains the FreeBlockList type, the FreeBlockNode type and its related routines. The FreeBlockList type is a linked-list type that can be used to keep track of free memory blocks in a memory pool while the FreeBlockNode type is a linked-node type to be used in conjunction with the FreeBlockList type.
