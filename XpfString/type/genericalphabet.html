GenericAlphabet

type, public, extends(BaseAlphabet) :: GenericAlphabet

The GenericAlphabet type is a generic data type that can represent any alphabets. An instance of the GenericAlphabet type can be created via the Construct method from either a given set of characters or a radix (representing a number of characters in the set where the characters' code are from 0 to radix-1).
